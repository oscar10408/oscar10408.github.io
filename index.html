Description

This project applies transformer-based temporal models to detect fine-grained GUI interactions (clicks, scrolls, inputs) in screen recording videos. It extends the ASFormer architecture and tailors it for GUI-specific use cases.

  • Detects keyframes from minimal visual changes
  • Improves segment accuracy with post-refinement
  • Enables downstream use cases like GUI agent training and tutorial summarization
